Students in a Class

January 2, 2015

Before starting here at Dev Bootcamp, I was a music teacher in NYC. The first week of every "mod" (as it was called) was tough, as any teacher will tell you. You have a room full of new kids, many of whom you have never seen before, and your job is to start connecting and instructing them right away. I thought it would make sense to make a nice little ruby program to help me remember the students in my class.

A good way to do this is to use ruby classes and class methods. A class is just what it seems, a group of objects with a similarity. They may all be Integers, Strings, or you can make one up. I decided to make a Students class, to include my students in.

class Student
end

jimmy = Student.new

Jimmy is the first student to be added to my Students class. I want to give each student some info so that I can remember who is who. A good way to do this is to initialize each student with attributes. You can do this using the initialize method, which is a special ruby class method. It will set those attributes to that Student when that student is created. I want to remember my students by some key characteristics: name, age, grade, instrument.

class Student

  def initialize(name, age, grade, instrument)
    @name = name
    @age = age
    @grade = grade
    @instrument = instrument
  end

end

jimmy = Student.new("Jimmy", 15, 9, "drums")

Now I have a student named "Jimmy", who is holding on to values that distinguish him. I want to be able to easily access those values, so that if I forget what he does I can find out easily. I'll create a method called student_info, which will give me all of the values, and then individual ones if I just want one value.

class Student

  def initialize(name, age, grade, instrument)
    @name = name
    @age = age
    @grade = grade
    @instrument = instrument
  end

  def student_info
      puts "#{@name} is a #{@age} year old #{@grade}th grader who plays the #{@instrument}."
  end

  def instrument
    puts "#{@name} plays the #{@instrument}."
  end

  def age
    puts "#{@name} is #{@age}."
  end

  def grade
    puts "#{@name} is in #{@grade}th grade."
  end

end

jimmy = Student.new("Jimmy", 15, 9, "drums")

jimmy.student_info => Jimmy is a 15 year old 9th grader who plays the drums.
jimmy.instrument => Jimmy plays the drums.
jimmy.age => Jimmy is 15.
jimmy.grade => Jimmy is in 9th grade.

At the end of your class, you have to write a teacher report on your students, so I found it useful to write some comments down, so that at the end of the year, you could look back and see how your students behaved, if they we're responsible, completed assignments, etc. So I found it beneficial to write comments about each student. A good way to do this is to use an attr_accessor method, which creates a readable and writable method for your instance variable. I'll call mine comments. 

class Student
  attr_accessor :comments

  def initialize(name, age, grade, instrument)
      @name = name
      @age = age
      @grade = grade
      @instrument = instrument
  end

  def student_info
    puts "#{@name} is a #{@age} year old #{@grade}th grader who plays the #{@instrument}."
      if @comments != nil
        puts "-----Comments-----"
        puts @comments
        puts ""
      end
  end

  def instrument
    puts "#{@name} plays the #{@instrument}"
  end

  def age
      puts "#{@name} is #{@age}"
  end

  def grade
      puts "#{@name} is in #{@grade}th grade"
  end

end


jimmy = Student.new("Jimmy", 15, 9, "drums")
jimmy.comments = "10/1/10: Sounded really good!  11/10/10: Showed up late to class"

jimmy.student_info => Jimmy is a 15 year old 9th grader who plays the drums.
    -----Comments-----
    10/1/10: Sounded really good!
     11/10/10: Showed up late to class

I'll include a couple more examples below of my whole class.

emily = Student.new("Emily", 17, 11, "piano")
jonathan = Student.new("Jonathan", 15, 9, "bass")
jonathan.comments = "Had a really good attitude"
claire = Student.new("Claire", 16, 10, "alto saxophone")
joseph = Student.new("Joseph", 18, 12, "trumpet")
austin = Student.new("Austin", 17, 12, "bass")

jonathan.student_info => Jonathan is a 15 year old 9th grader who plays the bass.
    -----Comments-----
    12/15/14: Had a really good attitude in class

joseph.student_info =>  Joseph is a 18 year old 12th grader who plays the trumpet.
austin.student_info  =>  Austin is a 17 year old 12th grader who plays the bass.

claire.instrument =>  Claire plays the alto saxophone

emily.grade => Emily is in 11th grade

And just like that, my class is easy to remember. Now I have no excuse for pointing and saying, "Hey you!"
