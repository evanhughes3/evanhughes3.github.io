How to make an array

    JS

  • var array = [1,"hello", true];
  • array.push("3") #=> 1,"hello", true, "3"
  • array.slice(1,3) #=> "hello", true, "3"
  • array[0] #=> [1]
  • array.indexOf("hello") #=> 1

  • array = [1,"hello", true]

Object / Hash

      
    
          
            
    JS Object
    
          
  • var greeting = {};
    • 
          
  • greeting['1'] = "Hello There!";
    • 
          
  • greeting['2'] = "How do you do?"; 
    • 
          
  • for (num in greeting) { console.log( num + " : " + greeting[num]); }
    • 
          
    
          
  • 1 : Hello There! 
    • 
          
  • 2 : How do you do?
    • 
              
          
            
    Ruby Hash
    
            
  • greeting = {}
    • 
            
  • greeting[1] = "Hello There!"
    • 
            
  • greeting[2] = "How do you do?"
    • 
            
  • greeting.each {|key, value| puts "#{key} : #{value}" }
  • 
            
    
            
  • 1 : Hello There!
    • 
            
  • 2 : How do you do?
    • 
              

          
    Objects in javascript and Hashes in ruby can be used in a similar way.  In a hash, you refer to a key and its value (1 => "Hello There!").  In an object, you refer to it's property, and that property's value (1: "Hello There!").  With a hash, you can use #each just like an array, just make sure to include |key, value| in your block.  In an object, use the for(x in array) syntax.
    
      

      
Objects / Class

        
    
          
            
    Object instantiate
          
  • function Animal(name, species, color, quote) {
  this.name = name;
  this.species = species
  this.color = color;
  this.quote = quote;
}

var tony_the_tiger = new Animal("Tony", "tiger", "orange", "They're GRRRRREAT!");

console.log(tony_the_tiger.name + " is a(n) " + tony_the_tiger.color + " "
+ tony_the_tiger.species + " who says '" + tony_the_tiger.quote + "'.")

Tony is a orange tiger who says 'They're GRRRRREAT!'.

    
          
    • 
          
            
    Class instantiate
    
          
  • class Animal

  def initialize(name, species, color, quote)
    @name = name
    @species = species
    @color = color
    @quote = quote
  end

  def description
    puts "#{@name} is a(n) #{@color} #{@species} who says #{@quote}."
  end

end

tony_the_tiger = Animal.new("Tony", "tiger", "orange", "'They're GRRRRREAT!'")
tony_the_tiger.description

Tony is a(n) orange tiger who says 'They're GRRRRREAT!'.

Objects in JS can be used like Classes in Ruby.  Above are examples of how create an instance of tony_the_tiger.  Assigning this.name = name, is similar to creating an attr_reader method in a ruby class.  

        
Randomization

        
    
          
            
    Javascript Randomization
    
          
  • var nums = [1,2, 3, 4, 5];

Math.random(); => 0.39196479227393866
Math.random() * nums.length => 1.9598239613696933
Math.floor(Math.random() * nums.length) => 1
nums[Math.floor(Math.random() * nums.length)] = nums[1] => 2

nums[Math.floor(Math.random() * nums.length)] => 5
nums[Math.floor(Math.random() * nums.length)] => 4

    • 

            
    Ruby Randomization
    

    nums = [1,2,3,4,5]

nums.sample => 4
nums.sample => 2


    
        

      
          
In Javascript, the random() method gives you a random decimal number.  To use that to randomize your array, you must multiply by number of items in array, then round it down to whole integer, which becomes your randomized index number.  The same function can be acheived using the built in #sample method in ruby.
